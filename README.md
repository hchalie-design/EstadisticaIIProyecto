# Proyecto Estadística II - Intervalo de Confianza con t de Student

## Descripción

Aplicación desarrollada en Java Swing para el curso de Estadística II de la Universidad Mariano Gálvez de Guatemala.

El programa permite calcular intervalos de confianza para la media poblacional utilizando la distribución t de Student (dos colas), mostrando de forma automática cada paso del procedimiento estadístico.

## Funcionalidades

- Cálculo de grados de libertad (gl).
- Obtención del valor crítico t.
- Cálculo del error estándar.
- Cálculo del margen de error.
- Determinación del límite inferior y superior.
- Cálculo automático del intervalo de confianza.
- Visualización gráfica de la distribución t de Student.
- Procedimiento paso a paso del cálculo.
- Interfaz gráfica amigable desarrollada con Java Swing.

## Datos de entrada

El usuario debe ingresar:

- Tamaño de la muestra (n)
- Media muestral (X̄)
- Desviación estándar (S)
- Nivel de confianza

## Tecnologías utilizadas

- Java
- Java Swing
- Apache Maven
- NetBeans IDE
